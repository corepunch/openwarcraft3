# Warcraft III Documentation

These notes collect the Warcraft III-specific file formats, renderer notes, UI/FDF behavior, and gameplay coverage work used by the main OpenWarcraft3 target.

The engine-wide client, server, networking, and runtime-module docs still live under the top-level `doc/architecture/` tree. The pages here are owned by the Warcraft III game target because they depend on Warcraft III assets, formats, scripts, data tables, or UI conventions.
